# 📊 Financial Performance Dashboard (Power BI)

## 🔍 Overview

This project is an end-to-end **Power BI dashboard** designed to analyze a company's financial performance using live, externally sourced data. The dashboard focuses on key business metrics such as revenue, profit, cash flow, growth rates, and shareholding patterns to provide actionable insights for decision-making.

The data is dynamically connected (not stored locally), ensuring that the dashboard reflects near real-time updates.

---

## 🎯 Objectives

* Track **Sales and Net Profit trends over time**
* Analyze **Cash Flow activities** (Operating, Investing, Financing)
* Evaluate **Profitability metrics** like ROE, CAGR, and Profit Growth
* Understand **Shareholding patterns**
* Identify **cost drivers** such as Interest and Depreciation
* Provide a consolidated view for **financial decision-making**

---

## 🛠️ Tools & Technologies

* Power BI (Data Modeling & Visualization)
* Power Query (Data Transformation)
* DAX (Measures & Calculations)
* Live Data Source (Online extraction)

---

## 📂 Data Model

The dashboard uses a structured data model with relationships between:

* **Quarterly Results** (Core financial metrics)
* **Profit & Loss**
* **Cash Flow**
* **Growth Metrics** (CAGR, ROE, etc.)
* **Date Table** (for time-based analysis)

The schema follows a relational structure enabling cross-filtering and time intelligence.

## 📊 Key Insights (High-Level)

* Sales show consistent growth, but profitability fluctuates
* Increasing costs (interest/depreciation) impact margins
* Cash flow inconsistencies highlight operational inefficiencies
* Shareholding trends indicate shifting investor confidence

## 🚀 How to Use

2. Refresh data to pull latest updates
3. Use slicers (Time Period, Date) to filter insights
4. Explore visuals to identify trends and patterns
